Detailed Description

template<template< concepts::FloatingPoint > typename ModelType, concepts::UnmodifiedFloatingPoint... Args>
struct wtf::detail_::ConstVariantFromTuple< ModelType, std::tuple< Args... > >

Implements ConstVariantFromTuple for std::tuple.

Template Parameters
ModelTypeA template template parameter that takes a type that satisfies the QualifiedFloatingPoint concept.
ArgsA parameter pack of types that satisfy the FloatingPoint concept.

This specialization is like the VariantFromTuple specialization, but results in each of the ModelType types to be const-qualified. See the description for the VariantFromTuple specialization for more details.
