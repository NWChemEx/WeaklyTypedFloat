Detailed Description

template<concepts::FloatingPoint FloatType>
class wtf::buffer::detail_::ContiguousViewModel< FloatType >

Type-erases a contiguous buffer of type FloatType*.

Template Parameters
FloatTypeThe type of floating-point value being held. Must satisfy the concepts::FloatingPoint concept.

This class aliases a contiguous buffer of floating-point values of type FloatType and models the type-erase interface defined by BufferViewHolder for the buffer.
