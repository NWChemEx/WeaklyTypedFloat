Detailed Description

Defines the interface between the type-erased and typed buffer.

This class provides FloatBuffer with a type-erased interface for interacting with the held buffer. Derived classes implement the actual typed implementations.

Are the elements in the buffer contiguous?

This method indicates whether the held buffer stores its elements contiguously in memory. If true, then it is safe to do pointer arithmetic off of the addresses of the typed elements. If false, then attempting to do arithmetic on the addresses will likely lead to undefined behavior.

Returns
