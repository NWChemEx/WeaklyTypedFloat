Detailed Description

Defines the interface used to interact with the type-erased float.

This class is the base class for all type-erased floating point holders. The FloatType class interacts with this class and then this class redirects the implementation to the typed derived class.

bool wtf::fp::detail_::FloatHolder::are_equal ( const FloatHolder &  other) const
inline

Polymorphic value comparison.

This method determines if the value held by *this is exactly equal to that held by other. The comparison is done polymophically, i.e., it will also ensure the state in the derived parts of *this and other are value equal. This method is implemented by calling are_equal_.

Note
The comparison will exactly compare the floating point values meaning they must be bitwise equal for built-in floating point types.
Parameters
[in]otherThe other FloatHolder to compare against *this.
Returns
True if *this and other are polymorphically equal and false otherwise.
Exceptions
NoneNo throw guarantee.
