Detailed Description

template<concepts::FloatingPoint FloatType>
class wtf::fp::detail_::FloatModel< FloatType >

Implements the FloatHolder API for floats of type FloatType.

Template Parameters
FloatTypeThe type of floating-point value being held. Must satisfy the concepts::UnmodifiedFloatingPoint concept.

This class is responsible for holding and knowing the type of a single floating-point value. It implements the FloatHolder API defined by the base so that the held value can be manipulated in a type-erased manner.

Provides mutable access directly to the held value.

Get/set are designed to be one-directional, i.e., you can't use get_value to also set the value. Sometimes we need to return the value in a modifiable way, e.g., when the user is unwrapping the wrapped value. Determines if *this is exactly equal to other.

Two FloatModels are exactly equal if the values they hold compare equal using operator==. Note that this is bitwise equality for built-in floating-point types (and should realistically be implemented similarly for user-defined types as well). This check will also require that the types of the objects being compared are the same.
