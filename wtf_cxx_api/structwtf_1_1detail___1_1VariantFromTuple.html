Detailed Description

template<template< concepts::FloatingPoint > typename ModelType, typename T>
struct wtf::detail_::VariantFromTuple< ModelType, T >

Primary template for creating a std::variant of ModelType types.

Template Parameters
ModelTypeA template template parameter that takes a type satisfying QualifiedFloatingPoint.
TA type that is expected to be a std::tuple of types satisfying FloatingPoint concept.

The primary template is undefined. A specialization is provided for when T is a std::tuple. The specialization is required to get at the types in the tuple. For each type U in T, the specialization should provide four member types:

  • value: A std::variant of ModelType* types
  • cvalue: A std::variant of ModelType<const U>* types
  • vvalue: A std::variant of ModelType<volatile U>* types
  • cvvalue: A std::variant of ModelType<const volatile U>* types
