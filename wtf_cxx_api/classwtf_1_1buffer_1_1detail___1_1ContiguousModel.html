Detailed Description

template<concepts::FloatingPoint FloatType>
class wtf::buffer::detail_::ContiguousModel< FloatType >

Type-erases a contiguous buffer of type FloatType*.

Template Parameters
FloatTypeThe type of floating-point value being held. Must satisfy the concepts::FloatingPoint concept.

This class models a contiguous buffer of floating-point values of type FloatType.

Compares the elements in the buffer for exact equality.

Value equal is defined as having the same elements in the same order. Same elements means operator== returns true for each corresponding pair. For most floating-point types this means bitwise equality.
