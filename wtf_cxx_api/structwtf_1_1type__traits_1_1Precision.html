Detailed Description

template<concepts::UnmodifiedFloatingPoint T>
struct wtf::type_traits::Precision< T >

Used to determine how many significant figures T has.

Template Parameters
TThe type to check. Must satisfy the UnmodifiedFloatingPoint concept.

The goal of this struct is to work out how many significant digits an object of type T can represent (in base 10). This is useful for trying to minimize precision loss in conversions among floating-point types.

This struct will contain a member value that is equivalent to std::numeric_limits<T>::digits10 when T is a type for which std::numeric_limits is specialized and 0 otherwise.

Users may specialize this class for their own types if they wish to have the precision detected correctly.
