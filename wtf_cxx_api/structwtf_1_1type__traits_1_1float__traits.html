Detailed Description

template<concepts::FloatingPoint T>
struct wtf::type_traits::float_traits< T >

Works out the types associated with T.

Template Parameters
TA floating-point type, possibly with cv-qualifiers.

Throughout WTF we will need more than just types like float, double, etc. We will also need references, pointers, and cv-qualified versions. Rather than assume that T is always a plain floating-point type, we use this traits class to do the conversion for us. Of note, this means that users can specialize the class for custom floating-point classes when those classes have custom reference/pointer classes.
