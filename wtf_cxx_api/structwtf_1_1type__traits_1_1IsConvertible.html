Detailed Description

template<concepts::UnmodifiedFloatingPoint From, concepts::UnmodifiedFloatingPoint To>
struct wtf::type_traits::IsConvertible< From, To >

Can an object of type From be implicitly converted to type To.

Template Parameters
FromThe type being converted from. Must satisfy the wtf::FloatingPoint concept.
ToThe type being converted to. Must satisfy the wtf::FloatingPoint concept.

This trait determines if an object of type From can be implicitly converted to an object of type To. This is determined by relying on the std::is_convertible type trait from the C++ standard library. If From can be implicitly converted to To then the static member value will be set to true, otherwise it will be false.
